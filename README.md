# Check Splitter

A small iPhone utility for splitting one restaurant bill quickly. Enter the numbers, see what everyone owes — on a single screen, no account needed.

## Features

- Enter subtotal, tax, tip, and number of people
- Tip as a dollar amount or percentage
- Instant equal-split result
- **Pro:** Round up each person's share to the nearest dollar
- **Pro:** One person pays extra — splits the remaining balance among the rest

---

## How It Works

1. Enter the subtotal (required) and any tax or tip
2. Set the number of people
3. The result card updates immediately — showing what each person pays
4. Use the Pro options for round-up or extra-payer scenarios
5. Tap Reset to start over

## Tech

- SwiftUI
- iOS 17+, iPhone only
- Local-first — no data leaves the device
